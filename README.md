Medical Imaging: Tumor Segmentation with Partial UNet++


Tıbbi Görüntüleme: Kısmi UNet++ ile Tümör Segmentasyonu


🇹🇷 Proje Hakkında (Turkish)
Bu proje, tıbbi görüntüler üzerinde derin öğrenme yöntemlerini kullanarak tümör segmentasyonu yapmayı amaçlamaktadır. Proje kapsamında, standart mimari üzerinde özelleştirmeler yapılarak Partial UNet++ (Kısmi UNet++) yapısı kullanılmıştır.

🚀 Öne Çıkan Özellikler
Partial UNet++ Architecture: UNet++ mimarisinin belirli katmanları ve derinlikleri, tümör segmentasyonu hedefine yönelik olarak optimize edilmiş ve kısmi olarak entegre edilmiştir.

Model Optimizasyonu: Mimarinin sadece belirli kısımlarının kullanılmasıyla işlem gücü verimliliği ve hedef bölgeye odaklanma başarısı artırılmıştır.

3D Segmentation: 3 boyutlu tıbbi veri setleri üzerinde hassas segmentasyon çalışmaları gerçekleştirilmiştir.

🛠️ Teknik Detaylar
Mimari: Modifiye edilmiş (Partial) UNet++.

Metrikler: Performans değerlendirmesi için Dice Skoru ve Hausdorff Mesafesi kullanılmıştır.

Platform: Google Colab Pro+.

🇺🇸 About the Project (English)
This project aims to perform tumor segmentation on medical images using deep learning methods. Within the scope of the project, a Partial UNet++ structure has been utilized by implementing specific customizations on the standard architecture.

🚀 Key Features
Partial UNet++ Architecture: Specific layers and depths of the UNet++ architecture have been optimized and partially integrated for the target of tumor segmentation.

Model Optimization: By utilizing only specific parts of the architecture, computational efficiency and focus on the target area have been enhanced.

3D Segmentation: High-precision segmentation studies have been conducted on 3D medical datasets.

🛠️ Technical Details
Architecture: Modified (Partial) UNet++.

Metrics: Dice Score and Hausdorff Distance were used for performance evaluation.

Platform: Trainings were conducted on Google Colab Pro+.
